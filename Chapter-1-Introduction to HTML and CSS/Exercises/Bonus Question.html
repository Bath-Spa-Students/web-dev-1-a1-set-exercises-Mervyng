HTML Tags

  • <h1> - <h6>

    Header tags for different levels of headings.

    <h1>Heading 1</h1>
  • <p>

    Defines a paragraph.

    <p>This is a paragraph.</p>
  • <ul> & <li>

    Creates an unordered list and list items.

    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
  • <a>

    Defines a hyperlink.

    <a href="https://example.com">Link</a>
  • <img>

    Embeds an image.

    <img src="image.jpg" alt="Description">
  • <div>

    Defines a division or section.

    <div>Content</div>

CSS Properties

  • color

    Sets the text color.

    color: blue;
  • font-size

    Sets the size of the font.

    font-size: 16px;
  • background-color

    Sets the background color.

    background-color: #f0f0f0;
  • margin

    Sets the margin space around an element.

    margin: 10px;
  • padding

    Sets the padding space around an element's content.

    padding: 5px;
  • border

    Sets the border around an element.

    border: 1px solid black;
  • text-align

    Sets the alignment of text.

    text-align: center;
  • display

    Defines how an element is displayed.

    display: block;
