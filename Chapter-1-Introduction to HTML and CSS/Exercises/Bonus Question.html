HTML & CSS Quick Reference Guide

A handy reference for common HTML tags and CSS properties

HTML Tags

  • <div>

    Defines a division or section in an HTML document.

    <div>
    Content goes here
</div>

  • <p>

    Defines a paragraph.

    <p>This is a paragraph</p>

  • <img>

    Embeds an image into the HTML document.

    <img src="image.jpg" alt="Description">

CSS Properties

  • color

    Sets the color of text.

    color: #333;

  • font-family

    Sets the font family for text.

    font-family: Arial, sans-serif;

  • background-color

    Sets the background color of an element.

    background-color: #f0f0f0;
